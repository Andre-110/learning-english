## 🚀 快速启动（3步）

### 步骤1：安装依赖
```bash
pip install -r requirements.txt
```

### 步骤2：配置API密钥
```bash
# 创建.env文件
cp .env.example .env
# 编辑.env，填入你的API密钥
```

### 步骤3：启动服务
```bash
# 方式1：使用启动脚本（推荐）
./start.sh

# 方式2：直接启动
uvicorn api.main:app --reload
```
