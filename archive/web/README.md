# LinguaCoach Web界面使用说明

## 功能特性

- ✅ **实时语音对话**：支持麦克风录音，实时语音转文字
- ✅ **流式音频输出**：支持WebSocket流式音频响应
- ✅ **实时评估**：显示语法、词汇、流利度、发音等评分
- ✅ **用户画像**：动态更新CEFR等级、优势、弱点、兴趣标签
- ✅ **对话记录**：完整的对话历史记录
- ✅ **学习报告**：对话结束后生成详细学习报告

## 使用方法

### 1. 启动后端服务

```bash
cd /home/ubuntu/learning_english
source venv/bin/activate
python -m uvicorn api.main:app --host 0.0.0.0 --port 8000
```

### 2. 访问Web界面

在浏览器中打开：
```
http://localhost:8000/
```

或者如果部署在服务器上：
```
http://your-server-ip:8000/
```

### 3. 开始对话

1. **输入用户ID**：在"用户ID"输入框中输入一个唯一的用户标识（例如：`user123`）
2. **点击"开始对话"**：系统会创建新对话并返回初始问题
3. **点击录音按钮**：开始录音，说出你的回答
4. **再次点击录音按钮**：停止录音，系统会自动处理并返回评估结果和下一个问题
5. **继续对话**：重复步骤3-4，进行多轮对话
6. **结束对话**：点击"结束对话"按钮，获取最终学习报告

## 两种模式

### HTTP模式（默认）
- 录音完成后，通过HTTP POST上传音频文件
- 服务器处理完成后返回完整响应（包括音频）
- 适合网络稳定的环境

### WebSocket流式模式（可选）
- 勾选"使用流式WebSocket模式"复选框
- 音频数据通过WebSocket实时传输
- 服务器可以流式返回响应
- 适合需要实时反馈的场景

## 界面说明

### 左侧控制面板
- **用户ID输入框**：输入用户标识
- **开始对话按钮**：创建新对话
- **结束对话按钮**：结束当前对话并获取报告
- **录音按钮**：开始/停止录音
- **状态显示**：显示当前操作状态

### 右侧对话面板
- **对话记录**：显示所有对话消息（用户和助手）
- **最新评估**：显示最近一轮的评分结果
- **用户画像**：显示当前用户的CEFR等级、优势、弱点、兴趣标签

## API端点

Web界面使用以下API端点：

- `GET /` - 提供Web前端页面
- `POST /voice-chat/start` - 开始对话
- `POST /voice-chat/{id}/speak` - 发送语音并获取回复
- `POST /voice-chat/{id}/end` - 结束对话
- `WebSocket /voice-stream/{id}` - 流式语音对话（可选）
- `WebSocket /realtime-speech/{id}/listen` - 实时语音输入（可选）

## 浏览器要求

- Chrome/Edge 88+（推荐）
- Firefox 85+
- Safari 14+
- 需要支持：
  - Web Audio API（录音）
  - WebSocket（流式通信）
  - MediaRecorder API（音频录制）

## 注意事项

1. **麦克风权限**：首次使用需要授权浏览器访问麦克风
2. **HTTPS要求**：某些浏览器在非HTTPS环境下可能无法访问麦克风
3. **音频格式**：浏览器通常使用WebM格式录制音频
4. **网络延迟**：语音识别和TTS生成需要一定时间，请耐心等待

## 故障排除

### 无法录音
- 检查浏览器是否已授权麦克风权限
- 检查麦克风设备是否正常工作
- 尝试使用HTTPS访问（某些浏览器要求）

### WebSocket连接失败
- 检查后端服务是否正常运行
- 检查防火墙设置
- 尝试使用HTTP模式代替

### 音频播放失败
- 检查浏览器音频设置
- 尝试刷新页面重新开始

## 技术栈

- **前端**：原生HTML/CSS/JavaScript
- **后端**：FastAPI + WebSocket
- **语音识别**：OpenAI Whisper / FunASR
- **语音合成**：OpenAI TTS / Edge TTS
- **AI模型**：GPT-4 / GPT-3.5-turbo
