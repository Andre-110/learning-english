# 用户说话录音存储位置说明

## 一、调试录音（Debug Recording）

### 存储位置
**服务器本地目录**：`{项目根目录}/debug_recordings/`

### 实现位置
- **前端**：`frontend/src/components/ConversationPanel.vue`（连接时自动录音，断开时上传）
- **后端**：`api/main.py` 中的 `POST /api/debug-recording`

### 文件名格式
```
debug-{userId}-{conversationId}-{timestamp}.webm
```
或（无会话 ID 时）：
```
debug-{userId}-{timestamp}.webm
```

### 技术细节
- 格式：`audio/webm;codecs=opus`
- 前端使用 `MediaRecorder` 录制，断开对话后通过 `FormData` 上传到 `/api/debug-recording`
- 服务器保存到 `debug_recordings/` 目录，便于在服务器上直接 debug

---

## 二、历史记录中的对话内容

### 存储位置
- **Supabase**（当 `STORAGE_BACKEND=supabase`）
- 表：`conversations`、`messages` 等

### 存储内容
- **仅文本**：用户消息、AI 回复的文本内容
- **不包含**：用户原始录音（.webm / PCM）

---

## 三、pVAD 噪音过滤

pVAD（Personalized Voice Activity Detection）用于过滤背景噪音/他人声音，仅在 `USE_PVAD=true` 时生效。

- **配置**：`USE_PVAD`、`PVAD_THRESHOLD`（见 `config/constants.py`）
- **说明**：需部署 ECAPA-TDNN 与 pVAD ONNX 模型（见 `services/pvad.py`），否则自动退化为直通

---

## 四、总结

| 类型           | 存储位置         | 格式 | 说明                       |
|----------------|------------------|------|----------------------------|
| 调试录音       | 浏览器下载目录   | .webm| 前端触发下载，未传服务器   |
| 对话历史       | Supabase         | 文本 | 仅消息文本，无录音文件     |
